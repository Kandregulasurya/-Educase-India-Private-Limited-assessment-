# PopX React Assignment

This project is a React.js implementation of the PopX mobile application UI based on the provided Adobe XD design.

## About the Project

The application consists of four screens:

* Welcome Page
* Login Page
* Create Account Page
* Account Settings Page

The design is responsive and follows the layout provided in the assignment. The mobile interface is centered on larger screens to match the original design.

## Features

* React.js based application
* React Router for page navigation
* Reusable Button and Input components
* Responsive design
* Local Storage support for storing user information
* Clean and organized folder structure

## Technologies Used

* React.js
* React Router DOM
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite

## Installation

Clone the repository:

```bash
git clone <https://github.com/Kandregulasurya/-Educase-India-Private-Limited-assessment->
```

Navigate to the project folder:

```bash
cd popx-react-assignment
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
